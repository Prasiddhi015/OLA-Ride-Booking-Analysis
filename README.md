# OLA Ride Booking Analysis

## Project Overview
This project analyzes OLA ride booking data for one month covering 
30,000 booking records. The goal was to find insights across 
cancellations, revenue, customer behavior and service quality.

## Dataset
- Source: OLA Ride Booking Data
- Period: April 2026
- Total Records: 30,000 bookings

## Dashboard Pages
1. Overall — Total bookings, booking trends, booking status
2. Vehicle Type — Vehicle wise booking and distance analysis
3. Revenue — Booking value, top customers, payment methods
4. Cancellation — Driver vs customer cancellations, reasons
5. Ratings — Customer and driver ratings by vehicle type

## Key Insights Found

**Cancellations**
- Overall cancellation rate is 25%
- Driver cancellations (18.02%) are 2.5x higher than customer (7.01%)
- Driver cancellations account for 72% of all cancellations

**Revenue**
- Total booking value is 8,465K
- E-Bike, Bike and Mini are the highest revenue generating vehicle types
- Average booking value per ride is 282

**Ratings**
- Average rating is 4.25 out of 5
- Bike is the highest rated vehicle by customers
- Ratings are consistent across all vehicle types between 4.0 and 4.4

**Customer Behavior**
- Top 5 repeat customers identified showing strong retention
- Some customers show consistent booking patterns

## Recommendations
1. Reduce driver cancellation rate through monitoring and penalties
2. Improve pickup and drop location accuracy
3. Increase availability of high demand vehicle types
4. Introduce loyalty programs for repeat customers
